A /now page — a snapshot of what has my attention this season, not an archive. If it looks stale, poke me.

Writing I'm actively working on

I'm building the essays as a deliberate arc — foundations first, then the bigger picture, then the AI layer on top:

  1. Payments foundations. Getting "How money moves" solid — authorize, clear, settle in plain language — then tracing a single coffee through every hop and fee. This is the bedrock everything else sits on.
  2. Structural shifts. Next in the queue: how the shape of the rails is changing, and what moves when it does. Currently an outline and a lot of margin notes.
  3. The AI layer. After that, where machine intelligence actually changes the economics — reconciliation, fraud, dispute triage — and where it's oversold.

Revisiting

I keep circling back to a geopolitics piece — "Who controls the rails?" — about what network access really means once you can be removed from it. It's a stub I'm not ready to publish because I want the public examples to be exactly right.

Learning & wandering

  • Reading up on how different countries actually settle money between banks — the unglamorous back-office that quietly decides how fast an economy moves.
  • Paying for things three different ways whenever I travel (cash, card, QR) and taking notes on where each one wins. Some of that lands in Travel.
  • Tending this garden itself — learning in public by rebuilding the site section by section.

If any of that overlaps with what you're chewing on, the door is open.
